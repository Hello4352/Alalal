📌 스팸 댓글 수집 프로젝트

이 페이지는 스팸 댓글 분류기 탐구 활동을 위해 댓글 데이터를 수집하는 공간입니다.
학생들이 직접 작성한 정상 댓글과 스팸 댓글을 모아 학습용 빅데이터를 구축합니다.

✏️ 댓글 입력 방법

  • 정상 댓글 칸과 스팸 댓글 칸이 따로 있습니다.
  • 각 칸에는 최대 3개의 댓글을 입력할 수 있습니다.
  • 댓글은 반드시 . (마침표)로 구분합니다.

예시: 오늘 수업이 유익했어요. 재미있었습니다. 다시 듣고 싶어요

🛑 스팸 댓글 정의

  1. 욕설·비속어 포함
  2. 주제와 무관한 댓글 (예: 과학 영상에 스포츠 이야기)
  3. 동일 단어나 문장이 3회 이상 반복
  4. 기존 댓글과 완전히 동일 (띄어쓰기 포함)
  5. 링크를 과도하게 포함
  6. 특수문자·이모지를 과도하게 포함
  7. 홍보성 문구 포함 (예: "구독해주세요", "이 링크 클릭", "돈벌기")
  8. 문맥과 무관한 자동생성 패턴 (예: "ㄴㅇㄹㄴㅇㄹ", "asdfgh")

📂 데이터 저장

입력된 모든 댓글은 GitHub Repository에 저장되며,
normal_comments.csv 와 spam_comments.csv 파일로 관리됩니다.
관리자는 모든 데이터를 직접 확인할 수 있습니다.

⚙️ 향후 활용

수집된 데이터는 전처리 후 머신러닝/딥러닝 모델 학습용으로 사용됩니다.
데이터셋과 분류기는 이 저장소에서 공개적으로 확인할 수 있습니다.
